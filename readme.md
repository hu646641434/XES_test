## 安装
### Installation with CocoaPods

1.在 `Podfile` 中添加 `pod 'XESnewtest'`。

2.执行 `pod install` 或 `pod update`。

3.导入 `"XESnewtest.h"`。

### 手动安装
1.下载XESnewtest文件夹内的所有内容。

2.将XESnewtest内的源文件添加（拖放）到你的工程。

3.导入XESnewtest.h。
